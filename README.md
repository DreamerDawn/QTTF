# QTTF 官网

欢迎来到 **QTTF 官网** —— 一个为丘班乒乓球联合会（QTTF）打造的轻量级数据展示平台。  
本项目目前处于**初期开发阶段**，旨在为乒乓球爱好者提供便捷的赛事信息、选手排名及相关内容。

## 🧠 关于本项目
- 本网页是 **Vibe Coding** 的杰作，全程由 [GitHub Copilot](https://github.com/features/copilot) 提供代码支持。
- 采用纯静态页面构建，托管于 GitHub Pages，访问流畅，开箱即用。
  
## ✅ 已完成页面
目前已完成以下三个页面的基础搭建：

| 页面 | 功能说明 | 链接（示例） |
|------|----------|--------------|
| **首页** (`homepage`) | 轮播展示 + 快速入口小部件 | [`/homepage`](https://dreamerdawn.github.io/QTTF/homepage) |
| **选手排名** (`rankings/latest`) | 最新选手排名表格（JSON 动态加载） | [`/rankings`](https://dreamerdawn.github.io/QTTF/rankings) |
| **关于我们** (`about`) | 组织简介、核心团队、加入方式等 | [`/about`](https://dreamerdawn.github.io/QTTF/about) |

## 🔧 技术栈
- **HTML5** + **CSS3**：语义化标签，Flex/Grid 布局，内嵌样式。
- **JavaScript (ES6)**：动态加载 JSON 数据、轮播交互、倒计时跳转。
- **Font Awesome**：图标库，用于箭头、社交图标等。
- **JSON**：选手排名数据独立存储，便于更新。
